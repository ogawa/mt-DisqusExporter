Disqus Exporter for MT
======================

This program is to convert MT comments to DISQUS comments.

INSTALLATION

1. Copy all the contents of this package into your MT directory.

2. Edit BLOG_ID, SHORT_NAME, ANONYMOUS_EMAIL, and USER_API_KEY
   constants in "tools/disqus-exporter.pl", so as to fit your
   environment.

3. Run "tools/disqus-exporter.pl" from the command line.
